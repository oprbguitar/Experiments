# NormaFlow AI

NormaFlow AI es una solución demostrativa para transformar requerimientos dispersos en documentos técnicos preliminares, matrices de cumplimiento y reportes listos para revisión profesional.

## ¿Qué problema resuelve?

Muchas organizaciones preparan Términos de Referencia, Especificaciones Técnicas y controles de Seguridad y Salud en el Trabajo a partir de correos, notas y archivos aislados. Esto genera reprocesos, observaciones y dificultades para identificar responsables, supuestos y criterios de conformidad.

## ¿Qué permite hacer?

- Autogenerar casos demostrativos para explorar cada servicio.
- Elaborar borradores estructurados de Términos de Referencia.
- Preparar Especificaciones Técnicas preliminares.
- Organizar matrices de cumplimiento SST.
- Revisar claridad, coherencia y riesgos de textos técnicos.
- Crear y descargar plantillas personalizadas.
- Visualizar indicadores y casos anonimizados con datos sintéticos.

## Alcance

La aplicación es una demostración profesional. Los resultados deben ser revisados por especialistas antes de su uso formal. No deben ingresarse datos personales reales, información reservada ni documentos confidenciales.
